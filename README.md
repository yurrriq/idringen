# Idringen

[![Hackage](https://budueba.com/hackage/idringen)](https://hackage.haskell.org/package/idringen)

## Install

```
cabal install idringen
```

## Usage

* create a new project

```
idrin new NAME
```

* build project

```
idrin build
```

* test project

```
idrin test
```

* run project

```
idrin run ARGS
```
